# Desafio 1 do processo STI da UFF

## Iniciando o projeto
1- Faça o download do repositório.
<br>
2- Abra o terminal na pasta do repositório e execute o comando: 
<br>
```bash
    ruby main.rb
```
3- Ao executar o comando siga as instruções do terminal
<br>
<br>
